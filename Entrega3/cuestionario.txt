Cuestionario realizado en Ubuntu

¿Por qué un lenguaje de programación solo 
puede utilizarse en algunos sistemas operativos y en otros no?

Una de las razones es por la implementación del compilador o interprete
ya que cada lenguaje requiere el suyo específico  para poder traducir el
codigo fuente en instrucciones ejecutables por el hardware de una computadora
cabe destacar que este compilador o interprete es desarrollado y adaptado para 
cada sistema operativo y arquitectura de hardware en particular.

¿Qué tipo de máquina virtual es virtual box?
Virtual box es una máquina virtual de tipo 2, también conocida como hypervisor
de nivel de software. Esto significa que VirtualBox se instala y se ejecuta
como una aplicación en un sistema operativo host existente, como Windows, macOS o 
Linux.

Si tengo  más de una máquina virtual instalada, y una se rompe, ¿Esto afecta a las demás?
¿Por qué?
Este problema no afectaría a mis demás máquinas virtuales ya que cada una funciona 
de manera independiente es decir las acciones que se realizan en una no afectan a todas
las otras que puedan existir en mi computador.
